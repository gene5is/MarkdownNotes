# 经济学十大原理

## 原理一：人们面临权衡取舍

**权衡取舍**：由于资源稀缺，人们必须在不同目标之间做出选择。例如：时间分配（工作 vs 休闲）、政府支出（国防 vs 福利）。

## 原理二：某种东西的成本是为了得到它所放弃的东西

**机会成本**：为了得到某种东西所必须放弃的最高价值的替代物。这是经济学中最重要的概念之一。

## 原理三：理性人考虑边际量

**边际分析**：理性决策者比较边际收益与边际成本。当边际收益大于边际成本时，决策者会采取行动。

## 原理四：人们会对激励做出反应

**激励**：诱使人们行动的因素，可以是正向的（奖励）或负向的（惩罚）。价格变化是重要的市场激励。

## 原理五：贸易可以使每个人的状况变得更好

**贸易收益**：通过专业化分工和贸易，每个人都可以消费更多的物品和服务。

## 原理六：市场通常是组织经济活动的一种好方法

**看不见的手**：市场价格引导自利的个体做出促进社会整体福利的决策。

## 原理七：政府有时可以改善市场结果

**市场失灵**：市场无法有效配置资源的情况，如外部性、公共物品、垄断。此时政府干预可以提高效率。

## 原理八：一国的生活水平取决于它生产物品与服务的能力

**生产率**：每个工人每小时生产的物品与服务数量，是生活水平的根本决定因素。

## 原理九：当政府发行了过多的货币时，物价上升

**通货膨胀**：经济中物价总水平的持续上升，通常由货币供应量增长过快引起。

## 原理十：社会面临通货膨胀与失业之间的短期权衡取舍

**菲利普斯曲线**：短期内，通货膨胀与失业存在此消彼长的关系。

# 微观经济学基本概念

## 稀缺性

**稀缺性**：社会拥有的资源是有限的，无法生产人们希望拥有的所有物品与服务。这是经济学的基本前提。

## 效率

**效率**：社会能从其稀缺资源中得到最大利益的特性。与效率相对的是**平等**，即经济成果在社会成员中公平分配的特性。

## 机会成本

**机会成本**：为了得到某种东西所必须放弃的最高价值的替代物。

## 理性人

**理性人**：系统而有目的地尽最大努力实现其目标的人。理性人在决策时会考虑边际量。

## 边际变动

**边际变动**：对行动计划的微小增量调整。理性决策者在边际收益大于边际成本时才会采取行动。

## 激励

**激励**：引起一个人做出某种行为的某种东西，如惩罚或奖励的预期。

## 市场经济

**市场经济**：当许多企业和家庭在物品与服务市场上相互交易时，通过他们的分散决策配置资源的经济。

## 市场失灵

**市场失灵**：市场本身不能有效配置资源的情况，原因包括外部性、公共物品、垄断、信息不对称等。

## 外部性

**外部性**：一个人的行为对旁观者福利的影响。正外部性（如教育）或负外部性（如污染）。

## 循环流量图

**循环流量图**：一个说明货币如何通过市场在家庭与企业之间流动的直观经济模型。

## 生产可能性边界

**生产可能性边界**：表示在可得到的生产要素与生产技术既定时，一个经济所能生产的两种产品数量的各种组合的图形。

## 实证分析与规范分析

**实证分析**：描述世界是什么的观点（描述性）。  
**规范分析**：描述世界应该是什么的观点（规定性）。


# 相互依存性与贸易的好处

## 基本概念

### 绝对优势

**绝对优势**：用比另一个生产者更少的投入生产某种物品的能力。

### 比较优势

**比较优势**：一个生产者以低于另一个生产者的机会成本生产一种物品的能力。贸易的好处来源于比较优势，而非绝对优势。

### 专业化分工

**专业化分工**：每个国家或个人专门生产自己具有比较优势的产品，然后通过贸易交换其他产品，从而提高整体经济效率。

### 贸易收益

**贸易收益**：通过专业化分工和贸易，双方都可以消费超出自己生产可能性边界的物品组合。

# 供给与需求

## 需求曲线

**需求**：消费者在一定时期内，在各种可能的价格水平下，愿意并且能够购买的商品数量。

**需求定律**：在其他条件不变时，一种物品的价格上升，对该物品的需求量减少。

**需求函数**： $Q_d = f(P, I, P_r, T, E)$

其中：
- $Q_d$ = 需求量
- $P$ = 价格
- $I$ = 收入
- $P_r$ = 相关商品价格
- $T$ = 偏好
- $E$ = 预期

**需求量的变动**：由商品价格变化引起的需求数量变化，表现为沿着需求曲线的移动。

**需求的变动**：由价格以外的其他因素变化引起的需求数量变化，表现为需求曲线的整体移动。

## 供给曲线

**供给**：生产者在一定时期内，在各种可能的价格水平下，愿意并且能够提供的商品数量。

**供给定律**：在其他条件不变时，一种物品的供给量与该物品的价格水平同方向变动。即一种物品的价格上升，该物品的供给量增加。

**供给函数**： $Q_s = f(P, C, T, P_i, E)$

其中：
- $Q_s$ = 供给量
- $P$ = 价格
- $C$ = 生产成本
- $T$ = 技术
- $P_i$ = 投入品价格
- $E$ = 预期

**供给量的变动**：由商品价格变化引起的供给数量变化，表现为沿着供给曲线的移动。

**供给的变动**：由价格以外的其他因素变化引起的供给数量变化，表现为供给曲线的整体移动。

## 市场均衡

**均衡**：市场价格达到使供给量等于需求量的水平。

**均衡价格**：使供给与需求平衡的价格，也叫市场出清价格。

**均衡数量**：在均衡价格下的供给量与需求量。

均衡价格和均衡数量是由供给曲线和需求曲线的交点决定的：

$$Q_d = Q_s$$

**过剩**：供给量大于需求量的状态。

**短缺**：需求量大于供给量的状态。

# 弹性理论

## 需求价格弹性

**需求价格弹性**：衡量需求量对价格变动的反应程度。

$$E_d = \frac{\%\Delta Q_d}{\%\Delta P}$$

| 弹性类型 | 弹性值 | 说明 | 价格与总收益关系 |
|---------|-------|------|------------------|
| 完全无弹性 | $E=0$ | 需求量完全不变 | 价格上升，总收益增加 |
| 缺乏弹性 | $0<E<1$ | 需求量变动比例小于价格变动比例 | 价格上升，总收益增加 |
| 单位弹性 | $E=1$ | 需求量变动比例等于价格变动比例 | 价格变动，总收益不变 |
| 富有弹性 | $E>1$ | 需求量变动比例大于价格变动比例 | 价格上升，总收益减少 |
| 完全弹性 | $E=\infty$ | 价格微小变动导致需求量极大变动 | 价格上升，需求量降为零 |

**影响需求价格弹性的因素**：
- 替代品的可得性
- 必需品与奢侈品
- 市场定义的宽窄
- 时间长短

## 需求收入弹性

**需求收入弹性**：衡量需求量对消费者收入变动的反应程度。

$$E_I = \frac{\%\Delta Q}{\%\Delta I}$$

- **正常品**： $E_I > 0$，收入增加，需求增加
  - **必需品**： $0 < E_I < 1$
  - **奢侈品**： $E_I > 1$
- **劣等品**： $E_I < 0$，收入增加，需求减少

## 需求交叉弹性

**需求交叉弹性**：衡量一种物品的需求量对另一种物品价格变动的反应程度。

$$E_{xy} = \frac{\%\Delta Q_x}{\%\Delta P_y}$$

- **替代品**：$E_{xy} > 0$，一种物品价格上升，另一种物品需求增加
- **互补品**：$E_{xy} < 0$，一种物品价格上升，另一种物品需求减少

## 供给价格弹性

**供给价格弹性**：衡量供给量对价格变动的反应程度。

$$E_s = \frac{\%\Delta Q_s}{\%\Delta P}$$

**影响供给价格弹性的因素**：
- 生产时间长短
- 生产要素的可替代性
- 生产规模的调整难度

# 消费者理论

## 效用与边际效用

**效用**：消费者从消费物品中获得的满足程度。

**总效用**：消费者从消费一定数量的商品中获得的总满足程度。

**边际效用**：消费者每增加一单位商品消费所获得的额外效用。

$$MU = \frac{\Delta U}{\Delta Q}$$

**边际效用递减规律**：随着消费数量增加，边际效用逐渐减少。

## 无差异曲线

**无差异曲线**：表示消费者偏好相同的两种商品的所有组合。

**特征**：
- 向右下方倾斜
- 凸向原点（边际替代率递减）
- 离原点越远，效用越高
- 任意两条无差异曲线不相交

**边际替代率**：消费者愿意用一种商品交换另一种商品的比率。

$$MRS_{xy} = \frac{\Delta Y}{\Delta X}$$

## 预算约束线

**预算约束线**：表示消费者在既定收入和商品价格下，能够购买的两种商品的所有组合。

**预算方程**： $P_x X + P_y Y = I$

其中：
- $P_x$ = X商品价格
- $P_y$ = Y商品价格
- $I$ = 收入

## 消费者均衡

消费者在预算约束下最大化效用的条件：

$$\frac{MU_x}{P_x} = \frac{MU_y}{P_y} = \lambda$$

其中 $\lambda$ 为货币的边际效用。

**替代效应**：由于商品相对价格变化引起的需求量变化。

**收入效应**：由于消费者实际收入变化引起的需求量变化。

# 生产者理论

## 生产函数

**生产函数**：表示在一定技术条件下，生产要素投入量与最大产出量之间的关系。

$$Q = f(K, L)$$

其中：
- $Q$ = 产量
- $K$ = 资本
- $L$ = 劳动

**短期生产函数**：至少有一种生产要素固定不变的生产函数。

**长期生产函数**：所有生产要素都可以调整的生产函数。

## 产量概念

**总产量(TP)**：一定投入下的总产出量。

**平均产量(AP)**：每单位可变要素的产量。

$$AP = \frac{TP}{L}$$

**边际产量(MP)**：每增加一单位可变要素所增加的产量。

$$MP = \frac{\Delta TP}{\Delta L}$$

**边际产量递减规律**：在其他投入不变时，随着一种投入的增加，边际产量最终会递减。

## 成本理论

**总成本(TC)**：生产一定产量所花费的全部成本。

$$TC = TFC + TVC$$

其中：
- **总固定成本(TFC)**：不随产量变化的成本（如厂房、设备）
- **总可变成本(TVC)**：随产量变化的成本（如原材料、劳动力）

**平均成本(AC)**：每单位产量的成本。

$$AC = \frac{TC}{Q} = AFC + AVC$$

其中：
- **平均固定成本(AFC)**：每单位产量的固定成本
- **平均可变成本(AVC)**：每单位产量的可变成本

**边际成本(MC)**：每增加一单位产量所增加的成本。

$$MC = \frac{\Delta TC}{\Delta Q}$$

**规模经济**：随着生产规模扩大，平均成本下降。

**规模不经济**：随着生产规模扩大，平均成本上升。

# 市场结构

## 完全竞争市场

**特征**：
- 大量买者和卖者
- 产品同质（完全替代）
- 自由进入和退出市场
- 信息完全
- 价格接受者

**总收益(TR)**：厂商出售产品所获得的全部收入。$TR = P \times Q$

**平均收益(AR)**：每单位产品的收益。$AR = \frac{TR}{Q} = P$

**边际收益(MR)**：每增加一单位销售所增加的收益。$MR = \frac{\Delta TR}{\Delta Q} = P$

**短期均衡**：厂商在短期内实现利润最大化的条件。$P = MR = MC$

**长期均衡**：厂商在长期内实现利润最大化且无超额利润的条件。$P = MR = MC = AC$

## 垄断市场

**特征**：
- 唯一卖者
- 产品无替代品或替代品极少
- 进入壁垒高（法律、技术、规模等）

**需求曲线**：向右下方倾斜，市场需求曲线就是厂商需求曲线。

**边际收益**：$MR < P$，因为垄断者必须降低价格才能销售更多产品。

**利润最大化**：$MR = MC$

**价格歧视**：垄断者对不同消费者收取不同价格，以获取更高利润。
- **一级价格歧视**：对每个消费者收取不同价格
- **二级价格歧视**：按购买数量收取不同价格
- **三级价格歧视**：对不同群体收取不同价格

## 垄断竞争市场

**特征**：
- 大量厂商
- 产品差异化（存在一定替代性）
- 自由进入和退出
- 厂商具有一定定价权

**短期均衡**：$MR = MC$，可能获得超额利润。

**长期均衡**：$MR = MC$ 且 $P = AC$，经济利润为零。

## 寡头市场

**特征**：
- 少数厂商（2-10家）
- 产品同质或差异化
- 进入壁垒较高
- 相互依存，策略性行为

**博弈论**：分析寡头厂商的策略互动。
- **囚徒困境**：个体理性导致集体非理性
- **纳什均衡**：给定其他参与者策略，每个参与者的最优策略
- **卡特尔**：厂商串谋以限制产量、提高价格
