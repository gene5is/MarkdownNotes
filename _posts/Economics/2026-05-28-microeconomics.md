# 经济学十大原理

- 人们面临权衡取舍

- 某种东西的成本是为了得到它所放弃的东西

- 理性人考虑边界量

- 人们会对激励做出反应

- 贸易可以使每个人的状况变得更好

- 市场通常是组织经济活动的一种好方法

- 政府有时可以改善市场结果

- 一国的生活水平取决于它生产物品与服务的能力

- 当政府发行了过多的货币时，物价上升

- 社会面临通货膨胀与失业之间的短期权衡取舍

# 供给与需求

## 需求曲线

**需求定律**：在其他条件不变时，一种物品的价格上升，对该物品的需求量减少。

**需求函数**：$Q_d = f(P, I, P_r, T, E)$

其中：
- $Q_d$ = 需求量
- $P$ = 价格
- $I$ = 收入
- $P_r$ = 相关商品价格
- $T$ = 偏好
- $E$ = 预期

**需求弹性**：衡量需求量对价格变动的反应程度

$$E_d = \frac{\%\Delta Q_d}{\%\Delta P}$$

## 供给曲线

**供给定律**：在其他条件不变时，一种物品的价格上升，该物品的供给量增加。

**供给函数**：$Q_s = f(P, C, T, P_i, E)$

其中：
- $Q_s$ = 供给量
- $P$ = 价格
- $C$ = 生产成本
- $T$ = 技术
- $P_i$ = 投入品价格
- $E$ = 预期

## 市场均衡

均衡价格和均衡数量是由供给曲线和需求曲线的交点决定的：

$$Q_d = Q_s$$

# 弹性理论

## 价格弹性

| 弹性类型 | 弹性值 | 说明 |
|---------|-------|------|
| 完全无弹性 | $E=0$ | 需求量完全不变 |
| 缺乏弹性 | $0<E<1$ | 需求量变动比例小于价格变动比例 |
| 单位弹性 | $E=1$ | 需求量变动比例等于价格变动比例 |
| 富有弹性 | $E>1$ | 需求量变动比例大于价格变动比例 |
| 完全弹性 | $E=\infty$ | 价格微小变动导致需求量极大变动 |

## 收入弹性

$$E_I = \frac{\%\Delta Q}{\%\Delta I}$$

- 正常品：$E_I > 0$
- 劣等品：$E_I < 0$
- 奢侈品：$E_I > 1$

## 交叉弹性

$$E_{xy} = \frac{\%\Delta Q_x}{\%\Delta P_y}$$

- 替代品：$E_{xy} > 0$
- 互补品：$E_{xy} < 0$

# 消费者理论

## 效用与边际效用

**效用**：消费者从消费物品中获得的满足程度。

**边际效用递减规律**：随着消费数量增加，边际效用逐渐减少。

$$MU = \frac{\Delta U}{\Delta Q}$$

## 无差异曲线

**无差异曲线**：表示消费者偏好相同的两种商品的所有组合。

**特征**：
- 向右下方倾斜
- 凸向原点（边际替代率递减）
- 离原点越远，效用越高

## 消费者均衡

消费者在预算约束下最大化效用：

$$\frac{MU_x}{P_x} = \frac{MU_y}{P_y}$$

# 生产者理论

## 生产函数

$$Q = f(K, L)$$

其中：
- $Q$ = 产量
- $K$ = 资本
- $L$ = 劳动

## 边际产量递减规律

在其他投入不变时，随着一种投入的增加，边际产量最终会递减。

## 成本理论

**总成本**：$TC = TFC + TVC$

**平均成本**：$AC = \frac{TC}{Q}$

**边际成本**：$MC = \frac{\Delta TC}{\Delta Q}$

# 市场结构

## 完全竞争市场

**特征**：
- 大量买者和卖者
- 产品同质
- 自由进入和退出
- 价格接受者

**短期均衡**：$P = MC$

**长期均衡**：$P = MR = MC = AC$

## 垄断市场

**特征**：
- 唯一卖者
- 产品无替代品
- 进入壁垒高

**利润最大化**：$MR = MC$

**价格歧视**：垄断者对不同消费者收取不同价格

## 垄断竞争市场

**特征**：
- 大量厂商
- 产品差异化
- 自由进入和退出

## 寡头市场

**特征**：
- 少数厂商
- 相互依存
- 策略性行为

**博弈论**：分析寡头厂商的策略互动
