# 宏观经济学概述

宏观经济学研究整体经济运行，关注经济增长、通货膨胀、失业率、国际贸易等问题。

# GDP核算

## GDP定义

**国内生产总值（GDP）**：某一既定时期内，一个国家或地区生产的所有最终物品与服务的市场价值。

## GDP核算方法

### 支出法

$$GDP = C + I + G + NX$$

其中：
- $C$ = 消费支出
- $I$ = 投资支出
- $G$ = 政府购买
- $NX$ = 净出口（出口 - 进口）

### 收入法

$$GDP = 工资 + 利息 + 租金 + 利润 + 间接税 + 折旧$$

### 生产法

$$GDP = \sum (各部门增加值)$$

## GDP的局限性

- 未包括非市场活动
- 未考虑环境成本
- 未反映收入分配
- 未衡量幸福程度

# 通货膨胀

## 定义与衡量

**通货膨胀**：经济中物价总水平的持续上升。

**消费者价格指数（CPI）**：衡量一篮子消费品和服务的价格水平。

$$CPI = \frac{当年一篮子物品的成本}{基年一篮子物品的成本} \times 100$$

**通货膨胀率**：

$$\text{通货膨胀率} = \frac{CPI_{t} - CPI_{t-1}}{CPI_{t-1}} \times 100\%$$

## 通货膨胀的类型

| 类型 | 通货膨胀率 | 说明 |
|-----|-----------|------|
| 温和通货膨胀 | < 10% | 价格缓慢上升 |
| 奔腾通货膨胀 | 10%-100% | 价格急剧上升 |
| 恶性通货膨胀 | > 100% | 价格失控 |

## 通货膨胀的成因

- **需求拉动型**：总需求超过总供给
- **成本推动型**：生产成本上升
- **结构性通货膨胀**：经济结构变化

## 通货膨胀的影响

- **皮鞋成本**：频繁往返银行的成本
- **菜单成本**：调整价格的成本
- **收入再分配**：损害固定收入者
- **不确定性增加**：影响投资决策

# 失业率

## 失业的衡量

**失业率**：

$$\text{失业率} = \frac{\text{失业人数}}{\text{劳动力人数}} \times 100\%$$

**劳动力参与率**：

$$\text{劳动力参与率} = \frac{\text{劳动力人数}}{\text{成年人口}} \times 100\%$$

## 失业的类型

- **摩擦性失业**：正常的职业转换
- **结构性失业**：技能不匹配
- **周期性失业**：经济周期导致

## 自然失业率

**自然失业率**：经济处于潜在产出水平时的失业率，等于摩擦性失业率与结构性失业率之和。

# 经济增长

## 增长核算

**生产函数**：

$$Y = A \times F(K, L)$$

其中：
- $Y$ = 产出
- $A$ = 全要素生产率
- $K$ = 资本
- $L$ = 劳动

**增长方程**：

$$\frac{\Delta Y}{Y} = \frac{\Delta A}{A} + \alpha \frac{\Delta K}{K} + (1-\alpha) \frac{\Delta L}{L}$$

## 趋同理论

- **绝对趋同**：穷国增长更快，最终赶上富国
- **条件趋同**：只有在相似条件下才会趋同

# 货币政策

## 中央银行工具

1. **公开市场操作**：买卖政府债券
2. **法定准备金率**：调整银行准备金要求
3. **贴现率**：调整央行贷款利率

## 货币乘数

$$\text{货币乘数} = \frac{1}{\text{法定准备金率}}$$

## 货币政策传导机制

$$\text{货币政策} \rightarrow \text{利率} \rightarrow \text{投资} \rightarrow \text{总需求} \rightarrow \text{产出}$$

## 货币政策规则

- **泰勒规则**：根据通胀和产出调整利率

# 财政政策

## 财政政策工具

- **政府购买**：直接支出
- **税收政策**：改变税率
- **转移支付**：社会保障等

## 乘数效应

**政府支出乘数**：

$$\text{乘数} = \frac{1}{1 - MPC}$$

其中 $MPC$ 为边际消费倾向。

## 挤出效应

政府支出增加导致利率上升，从而减少私人投资。

## 自动稳定器

- 累进税制
- 失业救济金

# 国际经济学

## 汇率

**名义汇率**：两种货币的兑换比率。

**实际汇率**：

$$\text{实际汇率} = \text{名义汇率} \times \frac{\text{国内价格}}{\text{国外价格}}$$

## 国际收支

**经常账户**：商品和服务贸易、收入转移

**资本账户**：资本流入流出

## 蒙代尔-弗莱明模型

分析开放经济下货币政策和财政政策的效果。

# 经济周期

## 周期阶段

1. **扩张期**：经济增长
2. **峰值**：经济最高点
3. **收缩期**：经济衰退
4. **谷底**：经济最低点

## 领先指标

- 股票价格
- 消费者信心指数
- 制造业PMI

# 宏观经济政策争论

## 相机抉择 vs 规则

- **相机抉择**：灵活应对经济变化
- **规则**：遵循固定政策规则

## 货币政策 vs 财政政策

- **货币政策**：影响利率和投资
- **财政政策**：直接影响总需求
