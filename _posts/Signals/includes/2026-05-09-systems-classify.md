# 系统分类

## 连续时间系统与离散时间系统

### 连续时间系统

连续时间系统处理连续时间信号，其输入和输出都是连续时间变量 $t$ 的函数。

**特点**：
- 输入输出均为连续时间信号
- 系统方程用微分方程描述
- 信号在整个时间轴上都有定义

**示例**：$RC$ 电路、模拟滤波器

### 离散时间系统

离散时间系统处理离散时间信号，其输入和输出都是离散时间变量 $n$ 的函数。

**特点**：
- 输入输出均为离散时间信号（序列）
- 系统方程用差分方程描述
- 信号仅在离散时刻有定义

**示例**：数字滤波器、计算机控制系统

## 即时系统（无记忆系统）与动态系统（记忆系统）

### 即时系统（无记忆系统）

即时系统的输出仅取决于同一时刻的输入，与系统过去的工作状态无关。

**数学定义**：
若系统满足 $y(t) = F[x(t)]$（连续）或 $y[n] = F[x[n]]$（离散），则称为即时系统或无记忆系统。

**特点**：
- 无储能元件
- 输出仅与当前输入有关
- 不需要初始条件

**示例**：电阻器 $v(t) = Ri(t)$

### 动态系统（记忆系统）

动态系统的输出不仅取决于当前输入，还与系统过去的状态有关。

**数学定义**：
若系统满足 $y(t) = F[x(t), x(\tau), \tau < t]$（连续）或 $y[n] = F[x[n], x[k], k < n]$（离散），则称为动态系统或记忆系统。

**特点**：
- 包含储能元件（电容、电感、寄存器等）
- 输出与当前输入和历史输入都有关
- 需要初始条件描述系统状态

**示例**：$RC$ 电路 $v_c(t) = \frac{1}{C}\int_{-\infty}^{t} i(\tau)d\tau$

## 线性系统与非线性系统

### 线性系统

线性系统满足齐次性和叠加性，即系统对多个输入的响应等于各输入单独作用时响应的代数和。

**数学定义**：

设系统对输入 $x_1(t)$ 的响应为 $y_1(t)$，对输入 $x_2(t)$ 的响应为 $y_2(t)$，则线性系统满足：

1. **齐次性（比例性）**：对任意常数 $a$，有
$$T\{a x_1(t)\} = a T\{x_1(t)\} = a y_1(t)$$

2. **叠加性**：有
$$T\{x_1(t) + x_2(t)\} = T\{x_1(t)\} + T\{x_2(t)\} = y_1(t) + y_2(t)$$

综合表示为：
$$T\{a_1 x_1(t) + a_2 x_2(t)\} = a_1 y_1(t) + a_2 y_2(t)$$

### 非线性系统

不满足齐次性或叠加性的系统称为非线性系统。

**非线性系统的特点**：
- 不满足叠加原理
- 可能包含非线性元件（如晶体管、变压器饱和）
- 分析方法与线性系统不同

**示例**：
- 乘法器：$y(t) = x_1(t) \cdot x_2(t)$
- 平方律器件：$y(t) = x^2(t)$
- 硬限幅器：$y(t) = \begin{cases} 1 & |x(t)| > 1 \\ x(t) & |x(t)| \leq 1 \end{cases}$

## 时变系统与时不变系统

### 时不变系统

时不变系统的特性不随时间变化。如果输入信号在时间轴上平移，输出信号也会有相同的平移。

**数学定义**：

若 $T\{x(t)\} = y(t)$，则时不变系统满足：
$$T\{x(t - t_0)\} = y(t - t_0)$$

**检验方法**：
1. 将输入 $x(t)$ 变为 $x(t - t_0)$
2. 求输出得到 $y_1(t)$
3. 将原输出 $y(t)$ 变为 $y(t - t_0)$ 得到 $y_2(t)$
4. 若 $y_1(t) = y_2(t)$，则为时不变系统

### 时变系统

时变系统的特性随时间变化，其参数或结构随时间改变。

**特点**：
- 系统参数随时间变化
- 不满足时不变性条件
- 常见于天线系统、时变滤波器等

**示例**：
- 调制器：$y(t) = x(t)\cos(\omega_0 t)$
- 变系数微分方程：$\frac{dy(t)}{dt} + t \cdot y(t) = x(t)$

## 因果系统与非因果系统

### 因果系统

因果系统的输出仅取决于当前和过去的输入，与未来输入无关。

**数学定义**：

若当 $t < t_0$ 时 $x(t) = 0$，有 $t < t_0$ 时 $y(t) = 0$，则系统是因果的。

即：$y(t_0) = F\{x(t), t \leq t_0\}$

**物理意义**：因果系统是物理可实现的，因为未来的输入还没有到来，不可能影响当前输出。

### 非因果系统

非因果系统的输出取决于未来时刻的输入。

**特点**：
- 输出与未来输入有关
- 非物理可实现（但可以在后处理中使用）
- 常见于图像处理、时间反转系统等

**示例**：
- $y(t) = x(t + 1)$（输出取决于未来输入）
- $y(t) = \frac{1}{2}x(t-1) + \frac{1}{2}x(t+1)$
- 反因果系统：$y(t) = x(t + 1)$（预测系统）

### 例子对比

| 系统 | 表达式 | 因果性 |
|------|--------|--------|
| 滑动平均 | $y[n] = \frac{1}{3}(x[n-1] + x[n] + x[n+1])$ | 非因果 |
| 累加器 | $y[n] = \sum_{k=-\infty}^{n} x[k]$ | 因果 |
| 理想低通滤波器 | $h(t) = \frac{\sin(\omega_c t)}{\pi t}$ | 非因果 |

## 集总参数系统与分布参数系统

### 集总参数系统

集总参数系统的电气特性可以由有限个集中分布的电路元件来描述。

**特点**：
- 系统尺寸远小于信号波长
- 可以用常微分方程描述
- 元件参数（$R$、$L$、$C$）集中在特定位置

**判断依据**：
若系统的几何尺寸 $d$ 满足 $d \ll \lambda$（信号波长），则为集总参数系统。

**示例**：$RC$ 电路、$RLC$ 电路

### 分布参数系统

分布参数系统的电气特性沿系统长度连续分布，必须用偏微分方程描述。

**特点**：
- 系统尺寸与信号波长相当或更大
- 必须用偏微分方程描述
- 电压和电流既是时间函数又是空间函数

**判断依据**：
若系统的几何尺寸 $d$ 与信号波长 $\lambda$ 满足 $d \geq \lambda/10$，则为分布参数系统。

**示例**：
- 传输线（长线）
- 天线
- 波导

### 长线方程

分布参数系统可用长线方程描述：

$$\frac{\partial v(z,t)}{\partial z} = -L\frac{\partial i(z,t)}{\partial t} - Ri(z,t)$$

$$\frac{\partial i(z,t)}{\partial z} = -C\frac{\partial v(z,t)}{\partial t} - Gv(z,t)$$

## 系统分类总结

| 分类标准 | 系统类型 | 特点 |
|----------|----------|------|
| 时间特性 | 连续时间系统 / 离散时间系统 | 信号类型 |
| 记忆特性 | 即时系统 / 动态系统 | 是否依赖历史输入 |
| 线性特性 | 线性系统 / 非线性系统 | 是否满足叠加原理 |
| 时变特性 | 时不变系统 / 时变系统 | 参数是否随时间变化 |
| 因果特性 | 因果系统 / 非因果系统 | 是否依赖未来输入 |
| 参数分布 | 集总参数系统 / 分布参数系统 | 尺寸与波长关系 |
