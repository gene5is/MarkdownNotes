# 模拟与数字滤波器

## 滤波器概述

### 滤波器的定义

滤波器是一种选频装置，允许特定频率的信号通过，同时抑制其他频率的信号。

### 滤波器的分类

**按频率响应分类**：
- 低通滤波器（LPF）：允许低频通过，抑制高频
- 高通滤波器（HPF）：允许高频通过，抑制低频
- 带通滤波器（BPF）：允许某一频段通过
- 带阻滤波器（BSF）：抑制某一频段
- 全通滤波器（APF）：所有频率都通过，但有相位变化

**按实现方式分类**：
- 模拟滤波器
- 数字滤波器

**按设计方法分类**：
- IIR滤波器（无限长单位脉冲响应）
- FIR滤波器（有限长单位脉冲响应）

### 理想滤波器

**理想低通滤波器**：

$$H(j\omega) = \begin{cases} 1, & |\omega| < \omega_c \\ 0, & |\omega| > \omega_c \end{cases}$$

**物理不可实现性**：理想滤波器的单位冲激响应是非因果的。

## 模拟滤波器

### 模拟滤波器的设计步骤

1. **确定技术指标**：通带截止频率、阻带截止频率、通带衰减、阻带衰减
2. **选择滤波器类型**：巴特沃斯、切比雪夫、椭圆等
3. **计算滤波器阶数**
4. **设计模拟原型滤波器**
5. **频率变换**（如果需要）

### 巴特沃斯滤波器

**幅度平方函数**：

$$|H(j\omega)|^2 = \frac{1}{1 + (\frac{\omega}{\omega_c})^{2N}}$$

**特点**：
- 最大平坦通带响应
- 单调下降的频率响应
- 过渡带较宽

### 切比雪夫滤波器

**切比雪夫I型**（通带等波纹）：

$$|H(j\omega)|^2 = \frac{1}{1 + \epsilon^2 C_N^2(\frac{\omega}{\omega_c})}$$

其中 $C_N(x)$ 为切比雪夫多项式。

**切比雪夫II型**（阻带等波纹）：

$$|H(j\omega)|^2 = \frac{1}{1 + \frac{1}{\epsilon^2 C_N^2(\frac{\omega_s}{\omega})}}$$

**特点**：
- 切比雪夫I型：通带等波纹，阻带单调
- 切比雪夫II型：通带单调，阻带等波纹
- 过渡带比巴特沃斯窄

### 椭圆滤波器

**幅度平方函数**：

$$|H(j\omega)|^2 = \frac{1}{1 + \epsilon^2 R_N^2(\frac{\omega}{\omega_c})}$$

其中 $R_N(x)$ 为雅可比椭圆函数。

**特点**：
- 通带和阻带都有等波纹
- 过渡带最窄
- 相位响应较差

### 贝塞尔滤波器

**特点**：
- 线性相位响应
- 群延迟近似恒定
- 幅度响应较差

## 数字滤波器

### IIR滤波器

**特点**：
- 无限长单位脉冲响应
- 递归结构
- 高效，但可能不稳定
- 非线性相位

**设计方法**：
- 脉冲响应不变法
- 双线性变换法

**脉冲响应不变法**：

$$h[n] = h_a(nT)$$

$$H(z) = \sum_{k=-\infty}^{\infty} H_a\left(j\frac{\omega + 2\pi k}{T}\right)$$

**缺点**：存在频率混叠。

**双线性变换法**：

$$s = \frac{2}{T} \cdot \frac{1 - z^{-1}}{1 + z^{-1}}$$

$$z = \frac{1 + \frac{T}{2}s}{1 - \frac{T}{2}s}$$

**特点**：无频率混叠，但有频率畸变。

### FIR滤波器

**特点**：
- 有限长单位脉冲响应
- 非递归结构
- 绝对稳定
- 可以设计线性相位

**设计方法**：
- 窗函数法
- 频率采样法
- 等波纹逼近法

**窗函数法步骤**：
1. 确定理想频率响应 $H_d(e^{j\omega})$
2. 计算理想单位脉冲响应 $h_d[n]$
3. 选择窗函数 $w[n]$
4. 计算实际单位脉冲响应 $h[n] = h_d[n] \cdot w[n]$

**常用窗函数**：

| 窗函数 | 主瓣宽度 | 旁瓣衰减 |
|--------|----------|----------|
| 矩形窗 | $4\pi/N$ | -13dB |
| 汉宁窗 | $8\pi/N$ | -31dB |
| 汉明窗 | $8\pi/N$ | -41dB |
| 布莱克曼窗 | $12\pi/N$ | -57dB |

**线性相位FIR滤波器**：

**条件**：$h[n] = \pm h[N-1-n]$

**类型**：
- 类型I：N为奇数，对称
- 类型II：N为偶数，对称
- 类型III：N为奇数，反对称
- 类型IV：N为偶数，反对称

## 滤波器的实现

### 模拟滤波器的实现

**无源滤波器**：使用电阻、电容、电感实现。

**有源滤波器**：使用运算放大器实现。

**典型电路**：
- Sallen-Key电路
- 无限增益多路反馈电路

### 数字滤波器的实现

**直接形式**：

$$y[n] = \sum_{k=0}^{M} b_k x[n-k] - \sum_{k=1}^{N} a_k y[n-k]$$

**级联形式**：

$$H(z) = A \prod_{k=1}^{L} \frac{b_{0k} + b_{1k} z^{-1} + b_{2k} z^{-2}}{1 + a_{1k} z^{-1} + a_{2k} z^{-2}}$$

**并联形式**：

$$H(z) = \sum_{k=1}^{L} \frac{A_k}{1 - p_k z^{-1}} + \sum_{k=1}^{M} \frac{B_k + C_k z^{-1}}{1 + D_k z^{-1} + E_k z^{-2}}$$

**格型结构**：适用于线性预测和自适应滤波。

## 滤波器的性能指标

### 时域指标

- 上升时间
- 超调量
- 稳态误差

### 频域指标

- 通带截止频率 $\omega_p$
- 阻带截止频率 $\omega_s$
- 通带波纹 $R_p$
- 阻带衰减 $A_s$

### 稳定性

**IIR滤波器稳定条件**：所有极点都在单位圆内。

**FIR滤波器**：总是稳定的。

## 滤波器的选择

### IIR与FIR的比较

| 特性 | IIR | FIR |
|------|-----|-----|
| 稳定性 | 需保证 | 绝对稳定 |
| 相位响应 | 非线性 | 可线性 |
| 计算效率 | 高 | 低 |
| 设计复杂度 | 中等 | 复杂 |
| 存储需求 | 小 | 大 |

### 选择准则

- 需要线性相位：选择FIR
- 对相位要求不高但要求高效：选择IIR
- 低阶实现：IIR更优
- 高阶、复杂响应：FIR更合适

## 滤波器设计流程总结

1. **需求分析**：确定技术指标
2. **类型选择**：IIR或FIR
3. **方法选择**：模拟原型或直接数字设计
4. **设计实现**：使用工具或手动计算
5. **验证测试**：检查频率响应、时域响应
6. **优化调整**：根据测试结果调整参数
